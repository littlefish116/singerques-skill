# 拟真漏斗：报名 → 三通电话 → 入选（逐关判定协议）

> 本文是「带闸门的拟真驱动器」——把节目组「问卷初筛 → 第一通 → 第二通 → 第三通(入选)」串成一条**逐关判定、决定是否放行下一关**的流程。
> 与 `interview_simulation.md` 分工：本文管**整条链 + 每关的通过/淘汰信号 + 判定尺度**；那篇管**每通电话里具体怎么应答**。
> **可信度图例**：`高` = 多人群、跨期反复印证；`中` = 个别过来人一手陈述；`低` = 孤立单一来源；`推测` = 群友猜测、未证实。

---

## 0. 诚实性边界（先读，避免误判）

漏斗的价值在"帮你校准自己卡在哪、怎么改"，**不在"保证入选"**。三点必须记住：

1. **筛选不是纯随机，是「随机抽看 + 投入度加权 + 年龄组竞争」**`中`：那四个文字题写得**越多、越用心**，**被注意到的概率越高**（"当论文写""每期基本不重复""填问卷90分以上可直接收一轮电话"）；同时**你的年龄组决定你跟多少人抢有限的配额**——500 名听审按 10+/20+/30+/40+/50+ 分 5 组、每组约 100 个投票位，**20+ 组公认最卷**（最有精力玩线下观演、报名最多），**40+/50+ 组节目组"很难摇到人"**、结构性更易入选。但**仍有运气地板**：哪怕写得好、年龄组占优，也可能没被那一轮抽到（"单纯就是你那份没被看到，是随机事件"；凌晨报名"疑似没被看过"）。
   > 换算成概率语言：**多写写好 + 年龄组占优，能显著抬高"被选中"的概率，但不能抬到 100%**——没被选中不叫"你不行"，叫概率没轮到。所以闸门0 的判定，看的是**你有没有把能控的概率尽可能抬高**，而不是"必中"。
2. **"电话之间晋级的精确阈值"语料偏薄**：群聊里"电话"相关消息绝大多数是情绪宣泄（"0电话""接到了"），真正分析"第一通怎样表现才能进第二通"的只有个别人概率体感（"第一通基本决定能否入选，约60%；第二通两天内约85%；第三通确认100%"`推测`）。所以**闸门0、闸门1 的"踩雷即淘汰"信号支撑硬（高/中）；闸门1→2→3 的表现阈值偏中/推测**——别把它当精确门槛。
3. **本漏斗不打包票**：我只判断"你这段呈现得更像/更不像会被放行"，**不替你保证"这样一定能入选"**（对齐 `SKILL.md` 禁止事项#7）。

> 一句话：把漏斗当**体检表**（指出弱项、给改法），不是**录取通知书**。

 (初筛+随机抽看+投入度加权)  (基础+态度+粉籍核查)  (深挖+一致性+独立性)  (确认档期+临场)
   踩雷=拉黑/0电话            踩雷=当场挂/拉黑        编造/矛盾=崩         档期冲突/乱说=掉

四道闸门逐级递进：**前一道过了，才有后一道的资格**；但任何一道踩中硬雷区（粉籍/多账号/纠缠），可能直接拉黑、后续"0电话"。

## 2. 交互判定协议（我如何陪你跑）

**入口（必做前置，不可跳）**：先按 `SKILL.md` 第 3 节「第 0 步」采集你的**真实**情况（听歌偏好/现场/动机/是否偏爱某在线歌手/看重什么，**外加年龄段 10+/20+/30+/40+/50+ 与职业**）。**漏斗判定建立在你的真实内容上，不替你编造。**

> 年龄/职业的用途（都如实说，不评判好坏）：
> - **年龄**：按「五年龄组配额」校准你的**入选概率**——20+ 最卷（要更出彩、随机地板最高），40+/50+ 节目组"很难摇到人"、结构性更易入选。**年龄只调概率，不改雷区判定**（粉籍/一致性/独立性对所有人一视同仁）。
> - **职业**：两件事——①若你是**音乐专业从业者**（学生/教师/制作人/歌手等），每期听审里都占**相当一部分**、属正常构成，但电话会沿"音乐经历 / 情感歌手 vs 技巧歌手"**深挖**（见 `answer_principles` 画像 C），如实专业地答即可；**其他行业是更大的一部分**，按普通听众主线即可。②节目组会确认你的**工作**（身份核查的一部分）；至于"能否到场、能否自担机酒"是它**直接问你**的、与职业无关——如实申报即可（⚠️ 不由职业推断经济能力，语料无此支撑）。

**逐关流程**（从闸门0开始，一道道走）：
1. 我扮节目组，按当关的典型动作出题 / 要材料（例：闸门0 要你交"问卷 5 题的草稿"；闸门1 出"对某在线歌手怎么看"）。
2. 你用自己的话作答（**别背稿**，允许停顿、不完美——背稿本身就是雷区，见 `risk_phrases` 第三节）。
3. 我按当关「判定维度」给**三档判定**之一 + 说明依据（引用语料信号）：
   - 🟢 **放行**：呈现到位、没踩硬雷，可进下一关。
   - 🟡 **暂缓（补强）**：方向对但有弱项/小风险——指出具体问题 + 给"更自然版本/补强建议"，改完可重判。
   - 🔴 **淘汰（本关）**：踩中硬雷区或编造被追崩——说明撞了哪条雷、为什么现实里会卡在这，**给复盘改法**（你下一轮报名能避坑）。
4. 你可随时要求**复盘某关**（"刚才那关我为什么是暂缓"）、或**跳到某关**单独练（"我就想练第二通二选一"）。

**判定尺度说明**：我按 `judge+coach`（判定+补强）模式跑——**判定严格照真实节目组口径**（粉籍、一致性、独立性真的会卡人），但**失败时一定给改法**，让你既练临场、又能学到东西。

## 3. 闸门 0：问卷 / 报名表 → 第一通电话

**作用**：初筛门槛 + 后续电话的"题源"（他们会顺着报名表问）。写得好**可能直接入选、跳过部分电话**；被不被看到是**「随机抽看 + 投入度加权」**——写得越多越用心，被注意到的概率越高。

**🟢 通过信号（抬高"被看到"的概率，更容易进第一通）**
- 报名表**被那一轮抽看到**——这是前提；**不是纯随机**：四个文字题**写得越多、越用心**（"当论文写""每期基本不重复""填问卷90分以上可直接收一轮电话"），**被注意到的概率越高**。`中`（投入度加权，多人印证）/ `推测`（90分直通）
- **年龄组竞争度**（第三维）：你是哪个年龄组，决定你跟多少人抢配额。**20+ 组最卷**（公认报名最多、最有精力）——你需要比平均更出彩、且随机地板最高；**10+/30+ 次之**；**40+/50+ 节目组"很难摇到人"**——你报名就比其他组更稀缺、结构性更易入选。`中`（五组配额 2024+2025 跨年印证 `中-高`；竞争排名用户一手 + 单帖印证）
- 写得**多元、多语种多风格**（写完摇滚写 rap、写完华语写欧美/日本），不每个答案堆同一位歌手。`中`
- **正脸、无美颜、与音乐挂钩**的生活照（演唱会/比赛/日常唱歌）。`中`（有亲历对比：侧脸未接、改正脸后入选）
- 论述题**多维度**作答（旋律/编曲/歌词/演绎），真诚 > 华丽。`中`

**🔴 淘汰/暂缓信号（卡在问卷关，0 电话）**
- **命中在线歌手的明显偏爱/唯粉痕迹**（报名表反复出现某一位）。`中`
- **多账号 / 重复报名**触发查重（"你有别的手机号吗"）。`高`
- **凌晨报名**的表"疑似没被看过"。`推测`
- 乐迷证明用**拼图 / 长截图**（2026 当季不要拼图、直接放单张照片）。`中`
- **一眼 AI 模板**（"一眼模板的回答节目组也看得出来"）。`中`

**判定维度（我怎么看你的问卷草稿）**：①四个文字题是否**写得充实**（抬高被看到概率的最直接杠杆）；②是否多元、不堆单一在线歌手；③论述是否有真实细节、经得起顺着问；④照片是否正脸+音乐相关；⑤有无拼图/多账号/AI味等硬伤；⑥**你的年龄组**——据此校准"你需要多出彩才算稳"（20+ 要更出彩、40+/50+ 容错更高），但不改变雷区判定。

> 现实校准：你把①②③④⑤做到、且年龄组占优，**被选中的概率会被显著抬高**，但仍有运气地板——可能还是没被那一轮抽到。这不代表你不行，是概率没轮到，下期重报即可。

## 4. 闸门 1：第一通电话 → 第二通电话

**作用**：基础核查 + 态度 + **粉籍核查**（第一通即查，可能远程指导你开权限看歌单/关注）。接到第一通≈"对你有意向"。

**🟢 通过信号（更容易进第二通）**
- 身份核对顺利、礼貌、手机畅通、环境安静。`中`
- 基础题（曲风/为什么来/怎么投）**多维度、不偏袒**，落点回"看现场综合表现"。`高`
- **第一通即配合粉籍核查**——公开各平台账号、远程开权限，坦然呈现**真实多元**审美（"听歌很杂、什么都有"）。`高（机制）`
- "面试不太过分都会过"。`推测`

**🔴 淘汰信号（当场挂电话 / 标记拉黑 / 之后0电话）**
- ⚠️ **表露对在线歌手的明确偏爱**（陷阱题："你对XX怎么看"→ 你说"我最爱ta"）→ **实习生当场挂电话**。`高`（陷阱/挂电话相关跨3群）
- ⚠️ **自爆参加过既往录制**→ "对方很快不和我多聊、挂了"。`高`
- **主页/歌单高度集中于在线歌手**（被审查到，粉籍明显）。`高`
- **多账号**被查。`高`
- **饭圈黑话 / 站队表态**（控评/打投/数据…）。`高`
- **过度内推 / 纠缠节目组**→ 拉黑，之后0电话。`高`

**判定维度**：①基础题是否多维不偏袒；②是否在粉籍核查下仍呈现多元（而非暴露偏爱）；③有无陷阱题踩雷（表露偏爱/自爆录过）；④礼貌与沟通是否正常。

> 🎓 **若你是音乐专业从业者**（每期听审占相当一部分）：这一关起可能被沿"你的音乐经历""情感歌手 vs 技巧歌手怎么看"**深挖**（见 `answer_principles` 画像 C、`question_bank` 第7题）。如实、专业地答即可——审美可信度高是优势，但别变"评委腔"、别替谁背书。

> 这是**最容易当场出局**的一关——硬雷区密集。重点练：被问在线歌手时给"有理由的欣赏+多元广度+落点公平投票"，而非"我最爱ta"。

## 5. 闸门 2：第二通电话 → 第三通电话

**作用**：深挖 + 一致性核查 + **独立性测试**（会不会被偏好/邻座/出场顺序带偏）。通常顺着第一通/报名表追问、设陷阱。

**🟢 通过信号（更容易进第三通）**
- 二选一**给条件、不站队**："看当期谁现场更适合那首歌"，选完补"另一位我也认可"。`高`
- "你会不会投喜欢的歌手？"→ **"反而更严"** + 举一个真实/合理的例子（"ta某次发挥一般我没投"）。`中`
- 深挖用**真实细节**展开（你备好的"后手"），且**与第一通/报名表一致**。`高`
- 独立性题明确"**只看当场现场表现，不受邻座/朋友/出场顺序影响**"。`中`（"两个歌手唱得差不多你投谁""朋友不喜欢某位你受影响吗"多帖印证）
- 涉及 **7选3 / 场景化公平**（1v1揭榜、慢歌、飙高音）时，用同一套标准、就事论事。`中`

**🔴 淘汰信号（两通后无下文）**
- **前后矛盾**：报名表写A、电话说B；或"听歌很杂"却只说得出一两位在线歌手。`高`
- **编造内容被追问崩**（一问"哪年/哪场"就卡壳、对不上）。`高`
- **站队/偏袒**在二选一中暴露。`高`
- "回答不流畅不完美也入选了"——说明这关**容错度其实不低**，关键是别矛盾、别踩雷。`中`

**判定维度**：①二选一是否给条件不站队；②深挖是否有真实细节且与前面一致；③独立性是否站得住（不受偏好/邻座/顺序影响）；④编造与否（追问下崩不崩）。

> 这是**最考验真实性**的一关——编的内容在这必崩。防崩唯一办法：以**真实**为单一信息源。

## 6. 闸门 3：第三通电话 → 入选

**作用**：确认意向 / 档期 / 能否到场 + 可能再问 1–2 个即兴题 + 通知结果。接到第三通基本≈入选。

**🟢 通过信号（拿到入选通知）**
- **档期如实确认**（能到场、能自担机酒与往返交通、提前预留档期）。`中`
- 即兴题当**音乐即兴评论**，自然、有观点、有分寸。`中`
- 礼貌收尾、表达感谢（不必过度）。`中`

**🔴 淘汰信号（到这关还掉）**
- 档期冲突 / 满口答应却放鸽子的风险。`中`
- 临场觉得"快通过了"就放松**乱说、与前面不一致**。`中`

**判定维度**：①档期是否如实可兑现；②即兴题是否自然有分寸、不与前面矛盾；③收尾是否得体。

> 这关通常较短、偏事务性——别因为快通过了就松懈，保持全程一致。

## 7. 元规则（贯穿整条漏斗）

- **通话时长 ≠ 结果**：有人 5–6 分钟即入选，有人聊 20+ 分钟却两通后无下文。`中` 别因电话短就慌、长就飘。
- **报名 → 入选有时间滞后**：并非"当周报名当周看"，可能被顺延到后续期。`中`
- **可重复报名**：当期没等到，可继续重新报名；但"当季去过一次后，该季基本不再选用"。`中`
- **落选判断**：直播当周**周三晚（外地报名者）还没接到电话 ≈ 当期落选**（仍可能被后续期顺延）。`中/低`
- **轮次口径**：核心是**三轮电话面试**；前后可能各有一次"确认通"（入选通知 / 录制当天确认抵达）。`中`
