# 总览：节目组筛选逻辑与你要呈现的形象

> 本文与同目录 `question_bank / answer_principles / risk_phrases / interview_simulation / faq` 配套使用。
> **可信度图例**：`高` = 多人群、跨期反复印证；`中` = 个别过来人的具体一手陈述、内部自洽；`低` = 孤立、单一来源、未被他人佐证；`推测` = 仅为群友猜测、未经证实（**凡猜测一律标"推测"**）。

---

## 0. 一条贯穿始终的原则：用"真实"打底

所有回答建议**必须建立在你自己的真实经历与真实观演动机之上**，本知识库提供的是**表达框架**，不是让你背诵的台词。

- 凡是涉及"你喜欢谁、听过什么、为什么想去"的地方，请用**你真实的情况**替换文中的 `〈…〉` 占位符。
- 不要编造没听过的歌手、没看过的现场、不存在的乐迷经历——电话那头会顺着你的回答追问（见 `question_bank` 追问题），编造很容易在第二、三通被戳穿。
- 你的真实动机（例如"喜欢现场不可复制的氛围""想亲耳听全开麦"）本身就是最好的答案，比任何"标准答案"都可信。

---

## 1. 总体筛选逻辑

节目组对大众听审的筛选，本质是在回答一个问题：**这个人会不会以粉丝身份带偏见投票？** 整个流程都是围绕"排除粉籍、保留理性听众"设计的。

| 环节 | 作用 | 可信度 |
|------|------|--------|
| **报名表 / 问卷初筛** | 第一道门槛；写得用心可能直接入选、甚至跳过部分电话。同时是后续电话提问的"题源"——他们会顺着报名表内容提问。 | 高 |
| **随机抽看** | 报名表被不被看到含较大运气成分；"不存在故意不选，单纯就是你那份没被看到"。 | 高 |
| **三轮电话** | 公认筛选流程（"大家都是通过三轮电话选出来的"）；每轮递进核查态度与一致性。 | 高 |
| **社交平台 / 听歌 App 审查** | 会看主页、歌单、关注（含 QQ音乐/网易云浏览），判断偏好与粉籍。**第一通电话即核查**：可能要求你公开各音乐/社交平台账号，甚至远程指导你打开权限以查看歌单、关注列表；还可能**让你在某首歌下留言核对账号、针对你发布的内容提问**（如"你发的歌曲风像某歌手，会不会偏好"），少数情况**让你当场唱歌**。 | 高（机制）/ 中（操作细节） |
| **查重 / 多账号识别** | 身份证+电话+姓名是稳定三要素；多账号、异常重复报名会被识别（"你有别的手机号吗"）。 | 高 |
| **拉黑机制** | 踩雷区、过度纠缠/内推节目组等可能被标记，之后"0 电话"。 | 高（机制）/ 中（具体触发线） |

> ⚠️ 关于"为什么会刷人"，群里有大量 `[推测]`：例如怀疑按专业分座、工作人员追韩流而选人、按 IP 判断账号真伪等——**均无实据，不要据此改变策略**，按"理性普通听众"主线准备即可。

## 2. 大众听审可能看重的特质

下列特质被多位入选者反复印证为"加分项"或"安全牌"：

- **无粉籍 / 不追星** —— "没有粉籍的才是最好的听审""我这种没有粉籍的可是稀缺资源"。`高`
- **能客观、综合地评价**，不预设立场 —— 看重唱功、编曲、舞美、现场氛围等多维度，而非"我就投某某"。`高`
- **真实的音乐热爱**，最好有可验证的"乐迷痕迹"（常听歌、看过现场、能说出所以然）。`中`
- **听歌风格多元（杂）** —— "我那一个歌单就一万多首，重金属摇滚爵士说唱流行都有"，显得不会为单一歌手投票。`中`
- **真诚、有礼貌、能正常沟通** —— 电话里"面试不太过分都会过"。`中`
- **回答前后一致**，不踩雷、不被追问出矛盾。`高`
- **有时间、有能力线下参与** —— 节目组会确认你的工作、所在城市、能否自行承担机酒与往返交通；需提前预留档期。`中`
- **形象得体、适合上镜** —— 节目会给观众镜头，现场观众多较注重仪表；上传生活照时留意此点（次要因素，不必焦虑）。`中/低`

## 3. 你应呈现的总体形象（一句话人设）

> **"一个听歌很杂、看重现场综合表现、没有粉籍、真诚礼貌的普通音乐爱好者。"**

把这个形象拆成可落地的"人设锚点"，电话前自己先想清楚（用你的真实情况填）：

1. **我是谁**：职业/身份（普通职业即可，不必音乐专业；非专业但热爱音乐反而是优势）。`〈填你的真实身份〉`
2. **我为什么想去**：一个**真实**的动机，如"现场全开麦的氛围电视里给不了""想亲眼看一次顶级现场"。`〈填你的真实动机〉`
3. **我怎么听歌**：风格多元，能说出 2–3 个不同类型的喜好（且不是本季在线歌手的"唯粉"）。`〈填你真实的曲风/歌手偏好〉`
4. **我怎么评价**：综合维度（唱功+编曲+舞美+氛围），有优有缺，不一边倒。`见 answer_principles`
5. **我和在线歌手的关系**：如实呈现你真实的多元审美即可；即使歌单/关注里有某位在线歌手，也坦然说明"只是听过几首、非追星"，**不必清理或隐藏账号**（伪装既不可取也未必有效，见 `SKILL.md` 禁止事项 #4）。`如实呈现多元审美`

> 这 5 个锚点是你**真实**情况的提炼，不是人设虚构。先把它们在脑子里/纸上理顺，面试时自然流露，远胜临场编造。
