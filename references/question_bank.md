# 题库：问卷 / 电话面试 / 追问 / 刁钻题 与考察意图

> 题目来自群内多位过来人的复述。**题型套路稳定**（"直播季三年接电话都问到过类似这样的问题，只不过稍微有一点不同"）`高`；但**每期具体歌手、具体曲目会变**，请用当季实际阵容替换。
> **可信度**：`高` = 题型被多人/跨期印证；`中` = 个别复述的具体题目；`推测` = 群友猜测。

---

## 一、问卷 / 报名表题目（初筛阶段）

报名表是电话提问的**题源**——他们会顺着报名表内容问，所以填什么 = 给自己挖什么坑。

### ⭐ 2026 当季问卷原文（5 题，至少答 3 题）`中`（用户提供的 2026 当季原文，与下方既有题型高度印证）

> 以下为 2026 当季问卷需**文字作答**的题目（个人信息、生活照、乐迷照因涉敏感/图像媒体，不在本库范围内）。**5 题至少答 3 题**。每题既是报名材料、也是后续电话的"题源"——填什么，电话就顺着问什么。
> ⚠️ **被看到的概率模型**：问卷筛选**不是纯随机**，而是「随机抽看 + 投入度加权」——这四个文字题**写得越多、越用心，被注意到的概率越高**（"当论文写""每期基本不重复""填问卷90分以上可直接收一轮电话"）；但仍有运气地板，详见 `screening_funnel.md` 闸门0。

**Q1. 推荐最近半年单曲循环最多的三首歌曲，并说出推荐原因（旋律、歌词、编曲、情感等）**
- 写法：三首**类型尽量不同**（别三首都是某一位歌手/同一曲风）；原因落到**音乐性维度**（旋律记忆点/编曲巧思/歌词意象/情感共鸣），别只写"好听"。
- 考察点：音乐审美是否多元、是否某歌手唯粉。
- 与电话的坑：电话会顺着问"为什么是这三首""其中某首的编曲好在哪"——填了就要答得出（防背稿）。

**Q2. 分享一场印象最深刻的音乐现场体验（场地、演出者、难忘瞬间）**
- 写法：**如实**写（真看过才写；没看过就跳过这题、答别的，别编——一问"哪年/哪场/印象"就崩，见 `risk_phrases`）。难忘瞬间给**具体细节**。
- 考察点：乐迷身份真实性、现场审美。
- 与电话的坑：顺着追问"那次最打动你什么"。

**Q3. 分享一场你没亲自到场、但通过视频/音频感受到"神级现场"的演出**
- 写法：选一个**真实看过录像、说得出所以然**的现场；多维度（Vocal/配器/舞美/现场 stable 度/氛围）。这一题**降低了"没看过现场"的门槛**——没去过现场的人也能答得扎实。
- 考察点：鉴赏力、能否多维度评价。
- 与电话的坑：可能引出"你心中的神级现场 vs 这一季某场，怎么比"。

**Q4. 如果入选大众听审，你的投票标准是什么？**
- 写法：标准答案方向 = **"综合现场表现（唱功/编曲/舞美/氛围/完成度）投票，不因个人喜好偏袒"**；可补"对喜欢的反而要求更严"。`高` 别写"我一定投某某"。
- 考察点：核心题——直接测你会不会带偏见投票。
- 与电话的坑：电话反复锚这道（"会不会投喜欢的""两个差不多投谁"），口径要前后一致。

**+ 乐迷证明图片**：**直接放单张照片，不要拼图/长截图**（2026 当季，见本节下文"乐迷证明"要点 + `cleaned_records` 信号）。体现乐迷身份即可，无则不强凑。

> 写法公约数：**真诚+用心+多元**，四个文字题尽量写充实（直接抬高被看到的概率）；自己写定稿（慎用 AI 润色）、前后一致、不堆砌单一在线歌手。详见下方"写法原则"。

> 📌 **题型演化（近年为准）**：2024 当季问卷为**四题全答**——①循环单曲+原因 ②最震撼现场+原因 ③最喜欢专辑+原因 ④最喜欢音综表演+原因（亲历者四题共写 600+ 字、多维度分析）；2026 演化为**五题（增"神级现场(音视频)"）至少答 3**。印证题型逐年演进，但核心考察点（审美多元 / 现场 / 投票标准 / 乐迷证明）稳定。`中`

| 常见栏目 | 写法要点 | 可信度 |
|----------|----------|--------|
| 喜欢的曲风 / 歌手 | 写**真实**偏好，但**避开对本季在线歌手的明显偏爱**；风格尽量多元。 | 中 |
| 为什么想来 / 观演动机 | 写真实动机（现场氛围、全开麦、想亲历顶级现场等），真诚 > 华丽。 | 中 |
| 看过哪些现场 / 乐迷证明 | 可附票根、照片证明乐迷身份（"我把我去的照片放报名表里了"）；没有也不必硬凑。 | 中 |
| 对往期节目/选手的评价 | 体现你能客观、多维度评价（不一边倒）。 | 中 |
| 评分标准 / 会怎么投 | 写"综合现场表现"类表述，别写"我一定投某某"。 | 中 |

**写法原则**`中`：
- **以自己写为主，慎用 AI** —— "我那些题目都是一个字一个字写出来的一点没用AI"是入选者原话；AI 味/套路感容易被识别，也容易在电话里对不上。来源有分歧，但**多帖亲历者允许适度 AI 润色**（"可把想法给 AI 润色、但必须真诚"），**稳妥口径：可用 AI 帮你理思路/润色语言，但最终文字务必自己写定稿**，确保被追问时答得出来、不像背稿。`中`
- **真实、用心即可**，不必堆砌；有人"随便写了几好听有感情"也入选了。也存在"把报名表当论文写"入选的案例——风格因人而异，**真诚+用心**是公约数。
- **前后一致**：报名表里写的内容，电话里要能对得上、能展开。
- ⚠️ **报名表里出现的在线歌手名要谨慎** —— 有反映"报名表出现过蔡依林和飞儿，命中两首歌，怕我有偏好被筛"。不必撒谎，但别主动堆砌某一位。`中`
- 📌 **照片**：用清晰**正脸、无美颜**照（有亲历者对比：传侧脸未接电话、改正脸后入选）；照片最好**与音乐挂钩**（演唱会/比赛/日常唱歌照），体现音乐素养。`中`
- 📌 **乐迷证明（2026 当季）**：**不要用拼图 / 长截图，直接放单张照片**即可。原始群聊有"我自从拼图就没被打过电话了""不喜欢那种长截图""不知道这次电话是拼图那次还是没拼图那次"等信号；小红书攻略多建议"拼图"，但**与当季实际冲突——以原始群聊为准**。`中`
- 📌 **论述题**：尽量**涉猎广**——写完摇滚写 rap/流行，写完华语写欧美/日本（多语种多风格），**不要每个答案都写同一位歌手**。`中`

## 二、电话面试常见题目

### 1. "对某位歌手 / 对强竞技模式怎么看？" `高`（题型）/ `中`（具体）
例："你对歌手这种强竞技模式怎么看？""问我对华晨宇有什么看法。"
- **意图**：看你是否有立场偏向；是否理性。
- **应对**：见 `answer_principles`——给有理由的看法，但**不吹捧、不贬低**，落到"看现场表现"。

### 2. "喜欢的曲风 / 平时听什么歌 / 喜欢的歌手？" `高`（题型）
- **意图**：判断你的音乐审美是否多元、是否某歌手唯粉。
- **应对**：说 2–3 个**真实**喜好且类型不同；如被追问"为什么"，能说出具体理由（这是防"背答案"的关键）。

### 3. "为什么想来歌手 / 推荐谁来？" `高`（题型）
- **意图**：看动机纯不纯；推荐谁也可能间接暴露偏好。
- **应对**：真实动机打底；推荐时给出**音乐性**理由而非"我偶像在"。

### 4. "对往期选手表现评价 / 评分标准 / 会怎么投？" `高`（题型）
例："他问了你投票标准""会不会因为更喜欢某种曲风，就投唱这类曲目的歌手？"
- **意图**：核心题——直接测你会不会带偏见投票。
- **应对**：标准答案方向 = "综合现场表现（唱功/编曲/舞美/氛围）投票，不因喜好偏袒"；可补"对喜欢的反而要求更高"。`高`

### 5. "有没有综艺/录制经历？去过哪一年？" `中`
- **意图**：核查身份、判断是否"老油条"或粉丝团常客。
- **应对**：**如实说**。注意：有反映"自爆参加过录制后，对方很快挂电话"（见 `risk_phrases`）。如实 ≠ 主动强调；问到再答，简短客观。

### 6. "这季舞台最喜欢哪个 / 看过哪些音综现场？" `中`
- **意图**：看你是否真在看节目、审美如何、是否有偏好。
- **应对**：说一个**真实**印象深的舞台，理由多维度，别只盯着某一位歌手。

### 7. （音乐专业向）"你的音乐经历 / 情感歌手 vs 技巧歌手怎么看？" `中`
- 仅当报名表体现音乐背景时可能被问；如实、专业地答即可。

### 8. 第二通·独立性测试："两个歌手唱得差不多，你这一票给谁？""朋友不喜欢某位歌手，你会受影响吗？" `中`（多帖印证）
- **意图**：测你会不会被周遭/偏好带偏，能否独立投票。
- **应对**：明确"**只看当场现场表现，不受邻座/朋友影响**"；"差不多的话，看那首歌谁更适合、谁完成度更高"。
- **相关机制**：投票为 **7 选 3**；还可能问**场景化公平**——1v1 揭榜、歌手唱慢歌、飙高音等场景下如何公正投票。`中`

### 9. "你为什么喜欢音乐 / 你和音乐的故事？" `中`
- 新近出现的开放式题，考察音乐热爱的真实性与表达。
- **应对**：用一个**真实**的小故事/细节切入，别喊口号。

### 10. "看过最近一期 / 哪几期？你会投给谁？说明原因。" `中`
- **意图**：核实是否真在看 + 独立判断（实为二选一陷阱的变体）。
- **应对**：落点仍是"看当场现场表现"，给**条件**不给站队（见第四节）。
- 变体："**上一期结果你认可吗？**""**你心目中的前三？**"——同样落"看当场现场表现"，**别因结果与自己预期不同就质疑公正**（质疑公正 ≈ 粉圈思维，是雷区）。`中`

### 11. "你对节目了解多少？看过前几季 / 本季吗？" `中`
- **意图**：核查是否忠实观众、真伪粉籍。
- **应对**：看过就如实讲真实印象；没怎么看就说没怎么看，别硬装资深。

### 12. ⭐ 2026 当季·AI / 科技题："怎么看 VR 演唱会 / AI 生成歌曲 / 虚拟歌姬（如洛天依）？" `中`
- **意图**：考察你对新音乐形态的态度与审美开放度（2026 新出现的题组）。
- **应对**：体现**理性 + 开放**——可真实表达倾向（如"AI 辅助编曲、提效是好事"），但别一味否定新形态；落点锚回"作为听审，我看重的是作品最终打动人的程度与现场完成度，而非它由谁/什么制作"。⚠️ 有 2026 落选者自评"死在 AI 这 part"（其对 AI 保守）——保守态度是否落选主因**待证**，但提示面对新事物宜体现理性判断而非立场站队。详见 `answer_principles` 第七节。

### 13. 二选一·新变体："唱功 vs 现场舞台表现力，更看重哪个？" `中`
- **应对**：给条件不给站队——"看那首歌更吃哪头：技术密集型曲目看唱功完成度，氛围/视觉型曲目看舞台表现力"，落点回"综合现场表现"。

### 14. "如何公平评价外国歌手和国内歌手？" `中`
- **意图**：测你会不会因语言/文化偏好带偏见。
- **应对**：用同一套标准（唱功/编曲/完成度/感染力），不因国籍加分或减分；可补"语言确实影响歌词共鸣，但现场音乐性是可比的"。

### 15. ⭐ "帮唱歌手 / 合作舞台会不会影响你的投票？" `中`（2026 二面新题，单帖亲历）
- **意图**：测你会不会因"喜欢某位帮唱嘉宾"就偏袒其合作的竞选歌手——仍是独立性 / 不站队的变体。
- **应对**：落点锚回**合作舞台的完成度与默契**。🌟 **真实成功范本（亲历者自评此答奠定出线）**："要看帮唱歌手跟竞选歌手的合作是否默契、能不能呈现一个更好的舞台、留下深刻作品，**而不是哪位帮唱嘉宾我喜欢就希望谁来**。"（看作品 / 默契，不站嘉宾）

> 一份较完整自述题库（原话）：① 喜欢什么类型曲风 ② 为什么想来歌手 ③ 推荐谁来 ④ 对往期选手表现评价 ⑤ 评分标准 ⑥ 平时喜欢听什么歌。`中`（个人复述，但覆盖面广）

## 三、追问题（顺着你的回答延伸）

面试官会**顺着你的回答继续问**——这是最考验真实性的环节。`高`

| 你的回答 | 可能的追问 |
|----------|-----------|
| "我喜欢 A 歌手" | "那 B 和 A 你选谁？""如果 A 和别人同场，你会投 A 吗？" |
| "我看过某现场" | "哪一年？说说印象？""那次最打动你什么？" |
| "我评分看综合" | "具体哪几维？举个上期的例子？""有没有哪次你给的分别人意外？" |
| 报名表写了某歌手 | 围绕该歌手深挖（"我看你报名表有尤长靖…"→ 一连串尤长靖相关） |

- **意图**：验证你是不是真懂、真看过；以及是不是粉丝。编造的内容在追问下最易崩。
- **应对**：每个回答都留一个**真实的、可展开的细节**当"后手"；不会的如实说"这个我没太关注"，别硬编。

## 四、刁钻 / 陷阱题

这些是**专门用来识别粉籍**的题，要特别小心：`高`（陷阱存在）/ `中`（具体措辞）

1. **"二选一"（强制选一位）** —— 例："尤长靖和艾略特你选谁？""参赛歌手里比较喜欢谁→接着问二选一。"
   - 陷阱：怎么选都可能被读成"粉某一位"。
   - 应对：**给条件、不给绝对站队**——"看当期谁现场发挥更好/谁更适合那首歌"，把选择标准落回"现场表现"而非"人"。实在要选，选完立刻补"但另一位的XX我也很认可"。
2. **"你会不会投你喜欢的歌手？"** —— 例："看你喜欢杰哥，如果选手选了他的歌你会投他吗？"
   - 陷阱：答"会"= 承认偏袒。
   - 应对：**"反而会更严格地要求他"**，举一个"喜欢的歌手某次发挥一般我没投"的真实或合理例子。`中`
3. **"你有别的手机号吗？"** —— 查多账号。
   - 应对：如实答（没有就说没有）；不要借号、多号报名。
4. **"你是不是 XX 的粉丝？"（直接质疑粉籍）** ——
   - 应对：**不慌、不辩解过度**；平静如实说"我听歌比较杂，XX 的歌我听过，但不算粉丝"，然后**把话题引回你的多元审美**。注意："当被怀疑的时候，怎么自证都没用"（群友感慨）`推测/中`——所以重点是**不留下可疑痕迹**（见 `risk_phrases`），而非临场自证。
5. **"你对上周节目怎么看？"（可能含当期争议）** ——
   - 应对：多维度、就事论事，不站队饭圈争议。

> 通用应对心法（被群友点破）：**"他们每次问我这种问题，我都是当即兴评述去说的"** —— 把它当成即兴的音乐评论，而不是答题；自然、有观点、有分寸。`中`

## 五、更多高频题与变体（补充）

题型套路稳定（"直播季三年都问到类似问题"）`高`，但具体歌手/曲目随当季变。下列为补充高频题与变体：

1. **"你怎么看这季的整体阵容 / 赛制？"** `中`
   - 意图：看你是否真在看、审美如何、有无立场。
   - 应对：就音乐论音乐，多维度评整体水平；不点名力挺、不贬低。
2. **"如果让你给某位歌手选一首歌，你会选什么？"** `中`/`推测`
   - 意图：偏好+曲风探测（选歌常暴露你偏爱谁/哪类风格）。
   - 应对：给**音乐性**理由（"这首歌的跨度适合 ta 的音色"），落到作品而非人。
3. **"你觉得这季谁可能走得更远 / 被淘汰？"** `中`
   - 意图：立场探测。
   - 应对：按"现场综合表现"判断，不点名"我看好/不看好 XX"；可说"还得看后续选曲和发挥"。
4. **"你手机里常用的音乐 App 有哪些？最近在听什么？"** `高`（审查听歌 App 确实存在）/ `推测`（是否当面问）
   - 意图：核实你的听歌习惯与多元度。
   - 应对：如实说 App 与最近真实在听的；体现"杂"。⚠️ 不要为了显得多元而编——他们真会去看。
5. **"你关注 / 喜欢本季哪位歌手吗？"** `高`（题型）
   - 意图：直接偏好探测。
   - 应对：**如实**——可以说欣赏某位，但立刻带出你真实的其他广度；落点"不会因喜欢就一定投"。
6. **"你看过去年（往季）的《歌手》吗？印象最深什么？"** `中`
   - 意图：真实性核查 + 审美。
   - 应对：看过就如实说一个真实印象（多维度）；没看过就说没怎么看，别硬编。
7. **"你为什么觉得自己适合当大众听审？"** `中`
   - 意图：自我认知、是否理解这个角色。
   - 应对：落"理性、听歌杂、能客观评价、看重现场综合表现"，用真实依据。

## 六、常见追问链（组合）

面试官常**连问**，提前想好链条的"下一环"：

- **偏好链**：喜欢谁 → 为什么 → ta 和别人你选谁 → 会不会投 ta → 给 ta 选首歌？
- **现场链**：看过现场吗 → 哪年/哪场 → 印象最深什么 → 那场你最打动的是哪一点？
- **标准链**：评分标准 → 具体哪几维 → 举个上期例子 → 有没有哪次给的分别人意外、为什么？
- **身份链**：有没有录制经历 → 哪年 → 怎么参与的 → 有别的手机号吗？

> 每条链的末端都指向**真实细节**。考前把自己的真实情况按这四条链各备 1 个真实"后手"，被追问就不慌。`中`
