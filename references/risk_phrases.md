# 高风险表达清单（雷区）

> 这里列的是**会让对方立刻判定你"有粉籍 / 在背答案 / 前后矛盾"**的表达与行为。
> 多条来自"踩雷后被挂电话/被刷/被拉黑"的真实反映。`高` = 多人印证；`中` = 个别案例；`推测` = 群友猜测。

---

## 一、高风险表达（直接触发警觉）

### 1. 表露对某位（尤其在线）歌手的明确偏爱 `高`
- "我掉进陷阱题，说了对某个首发歌手的喜欢，结果实习生立马挂电话。" `高`
- 雷区句式：
  - "我是 XX 的粉丝 / XX 是我最爱的歌手"
  - "我专门来看 XX 的 / 我就是为了 XX 来的"
  - "XX 一定夺冠 / 我必投 XX"
- **替代**：可以真诚欣赏，但要落到"看现场当期表现"，并搭配对其他人的认可。

### 2. 自爆参加过既往录制 `高`（机制）/ `中`（个案）
- "我回答参加过之前歌手录制后，实习生很快就不和我多聊、不继续问就挂了。"
- **应对**：问到才如实简短答，不主动强调、不渲染。

### 3. 报名表/歌单/关注"命中"在线歌手 `中`
- "报名表出现过蔡依林和飞儿，命中两首歌，怕我有偏好被筛。"
- "我把王菲也删掉了，听说接线员问组长'王菲行不行'，说不行。"
- **应对**：报名前清理在线歌手的明显追星痕迹；保留的如实说明"只是听过几首"。

### 4. 多账号 / 重复报名 / 借号 `高`
- 被问"你有别的手机号吗？""是不是有两个账号？"
- **应对**：一个身份一个号，不要借号、多号报名（有人借的号是流量卡不能接电话，直接误事）。

### 5. 过度内推 / 纠缠节目组与实习生 `高`
- "越热爱节目组、内推花样搞尽，反而把自己搞被拉黑了。"
- "两次内推没通过就把我和好几个人拉黑了。"
- **应对**：正常报名、正常接电话即可；别反复托关系、骚扰工作人员。

### 6. 饭圈黑话与站队表态 `高`
- 控评 / 打投 / 数据 / 超话 / 应援 / 塌房 / 唯粉 / CP / 拉踩…——任何一句都会暴露饭圈属性。
- 直接贬低某歌手、或为某歌手"撕"别人。

## 二、容易被认为"粉籍明显"的表达

凡让正常人觉得"这人明显是某歌手粉丝"的，都算：

- 对某一位歌手**如数家珍、滔滔不绝**，对其他人**一问三不知**。
- 反复把话题**引向同一位歌手**。
- 评价时**情绪浓度过高**（激动、护短、替其鸣不平）。
- 社交主页/歌单**高度集中于在线歌手**（会被审查到）。
- 用"我们家 / 哥哥 / 姐姐"等粉圈称呼。 `中`

> 判断口径：如果你的回答让一个陌生人 3 秒内能说出"你是 XX 粉丝"，那就是粉籍明显。目标是让对方**说不出你具体粉谁**。

## 三、容易显得"在背答案"的表达

背答案 = 不可信，且与"理性真实听众"人设冲突：

- 回答**过于工整、像作文**，每句都对仗、没口语停顿。
- 用**网络模板句**（"始于颜值、陷于才华""XX 的声音有故事"等烂大街话术）。
- 对每个问题都**秒答、长篇大论**，没有"嗯…我想想""这个我不太确定"等真人语态。
- **用 AI 生成**报名表/回答（"我一个字一个字写出来的一点没用AI"——反例说明真人手写是加分项）。 `中`
- 细节**经不起追问**——一问"具体哪期/哪个现场"就卡壳或前后对不上。

> **替代**：用即兴评述的语气（"他们每次问我，我都是当即兴评述去说的"），允许停顿、允许不完美、允许"我记不太清了"。`中`

## 四、容易"前后矛盾"的表达（电话里被追问崩）

三轮电话之间、报名表与口述之间，**不一致 = 大忌**：

- 报名表写"喜欢 A"，电话里说"最喜欢 B"。
- 第一通说"没看过现场"，第二通又说"去年去过"。
- 说"听歌很杂"，但被追问具体曲风/歌手时说不上来，或只说得出一两位在线歌手。
- 评分标准前后不一：一会儿"看唱功"一会儿"看感情"。
- "如实"与"清理痕迹"打架：嘴上说没关注某歌手，主页却全是 ta。

> **防矛盾唯一办法**：以**真实**为单一信息源。电话前把 `overview.md` 的 5 个锚点（身份/动机/曲风/评价方式/与在线歌手关系）用你的**真实情况**写一遍、记熟，三轮电话都照这个讲，自然不会前后打架。编的内容才需要记"上次撒了什么谎"——真话不用记。
